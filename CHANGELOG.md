# Changelog

## [2.6.0] — 2026-03-22

### Админ-панель: редизайн дашборда

- Новый дашборд с 4 стат-карточками: Ноды, Клиенты, Активные прокси, Онлайн
- Стат-карточки с цветными иконками, градиентной полоской при ховере и анимациями
- Карточки нод переведены на CSS-классы `node-card` с флагами стран, статистикой и ховер-эффектами
- Убраны избыточные inline-стили, всё переведено на дизайн-систему

### Админ-панель: дизайн и UX

- Глобальный размер SVG-иконок (16px) — исправлены огромные иконки в карточках
- Исправлено мерцание страницы «Прокси» при переходе (loadedOnce ref)
- Мобильное меню-гамбургер с оверлеем и закрытием при навигации
- Улучшенный логин: анимация `login-pop`, пульсация иконки
- Боковая панель: градиентный фон, ховер на логотипе
- Модалки: тонкая фиолетовая рамка-тень
- Тосты: `backdrop-filter: blur(12px)`
- Мобильная панель: glassmorphism-эффект, glow активной иконки
- Пустые состояния: увеличенные иконки и улучшенная типографика

### Админ-панель: исправления

- Исправлен Toast API в DatabaseManager, AnnouncementsManager, BackupsManager
- Добавлены CSS-переменные `--primary`, `--danger`, `--ac`, `--bg4`
- Добавлен CSS-класс `btn-warning`
- Исправлена опечатка `className='input'` → `'form-input'` в AnnouncementsManager

### Backend

- Исправлен баг аллокации портов в `createRemoteUser` — сканирование max port из docker-compose файлов вместо подсчёта директорий

## [2.5.0] — 2025-07-23

### Агент v2.0

- Полностью переписан `mtg-agent` (Python FastAPI) — v2.0
- Системные метрики: CPU, RAM, диск, аптайм, load average через `/proc`
- Per-container ресурсы: CPU%, память, аптайм контейнера
- Кэширование метрик (15 сек) с фоновым обновлением — мгновенные ответы
- Управление контейнерами через агент: restart, stop, start
- Получение логов контейнеров через агент
- Docker Compose: healthcheck, JSON-логирование, ограничение размера логов
- Обновлённый `install-agent.sh` для v2.0

### Админ-панель: метрики нод

- Карточка системных метрик на странице ноды (CPU, RAM, Диск, Аптайм)
- Кнопки управления контейнерами: перезапуск / остановка / запуск
- Модалка просмотра логов контейнера (последние 200 строк)
- Прогресс-бары RAM/Диск с цветовой индикацией (зелёный/жёлтый/красный)

### Backend

- Новые эндпоинты: `/api/nodes/:id/system`, `/api/nodes/:id/full-metrics`
- Управление контейнерами: `POST /api/nodes/:id/containers/:name/:action`
- Логи контейнеров: `GET /api/nodes/:id/containers/:name/logs`
- `/api/status` теперь возвращает системные метрики нод
- `ssh.js`: поддержка POST в `agentFetch`, 5 новых функций

## [2.4.0] — 2026-03-19

### Баланс клиента

- Баланс отображается на дашборде — отдельная карточка с градиентным фоном
- Пополнение баланса через ЮКассу — модалка с предустановленными суммами (100 / 200 / 500 / 1000 ₽) и произвольным вводом
- Оплата заказа с баланса — кнопка «Оплатить с баланса» в модалке подтверждения (если средств достаточно)
- Оплата через ЮКассу по-прежнему доступна как альтернативный способ
- Авто-продление списывает средства с баланса и продлевает прокси автоматически
- Новые API-эндпоинты: `GET /balance`, `POST /balance/topup`, `POST /orders/:id/pay-balance`

### Админ-панель: управление .env

- Настройки `.env` файла прямо из админ-панели (раздел «Настройки»)
- Редактирование всех переменных окружения: порт, JWT, YooKassa, SMTP, Telegram, НПД
- Перезапуск сервера после сохранения изменений

### Админ-панель: управление прокси клиентов

- Создание и удаление прокси для клиента прямо из карточки клиента
- Выбор ноды и автоматическая настройка прокси

### Версия и обновления

- Отображение git commit hash в модалке версий / обновлений
- Сборка клиентского SPA теперь выводится в `backend/public-client/`

## [2.3.0] — 2026-03-19

### Повтор платежа

- Добавлена кнопка «Повторить» для отменённых платежей в истории платежей
- На странице результата оплаты при ошибке появилась кнопка «Повторить оплату»
- Новый API-эндпоинт `/payments/:id/retry` — автоматически создаёт новый заказ и платёж
- При повторе сохраняются тариф, локация и все параметры исходного заказа

### Плавная загрузка сайта

- Добавлен splash-экран с анимацией при загрузке приложения
- Плавное исчезновение загрузчика после инициализации React
- Анимация fade-in для содержимого приложения при первом рендере

### Удаление прокси

- Клиенты теперь могут удалять ненужные прокси из списка «Мои прокси»
- Модальное окно подтверждения с предупреждением для активных прокси
- При удалении активного прокси он автоматически отключается на сервере
- Новый API-эндпоинт `DELETE /orders/:id` с полной очисткой (сервер + БД)

## [2.2.0] — 2026-03-18

### Навигация и UX

- Авторизованные пользователи теперь могут вернуться на главную страницу сайта
- На главной отображается кнопка «Личный кабинет» вместо «Войти / Регистрация» для авторизованных
- Добавлена ссылка «Главная» в футер личного кабинета

### Пинг серверов

- Добавлено отображение пинга (задержки) для каждого прокси-сервера в дашборде
- Пинг отображается в детальной карточке прокси с цветовой индикацией (зелёный/жёлтый/красный)
- Новый API-эндпоинт `/proxies/:orderId/ping` для измерения задержки до сервера

### Подтверждение заказа

- Добавлено модальное окно подтверждения перед оплатой на странице тарифов
- В окне отображается сводка: тариф, период, локация, количество устройств и итоговая сумма
- Двухшаговый процесс: «Оформить заказ» → подтверждение → «Перейти к оплате»
